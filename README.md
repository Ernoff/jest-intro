# Introduction to Testing with Jest, Nock and Sinon

An example of unit testing and integration testing with Jest, Nock and Sinon

## Getting Started
---
  
* To execute this script, first install all dependencies by running:

      npm install

* After all the dependencies has been installed, make script executable by running

      chmod +x index.js

* To execute the script run

      ./index.js

* To run test, run

      npm test
